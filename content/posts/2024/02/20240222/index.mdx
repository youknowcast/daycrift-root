#### 海南鶏飯

出典不明，手元に手書きのレシピだけあるので転載する．

**材料**

| 品目        | 量    | コメント                   |
|-----------|------|------------------------|
| 鶏もも肉      | 一枚   |                        |
| 米         |      |                        |
| 醤油        |      |                        |
| 料理酒       |      |                        |
| 生姜        | 適量   | チューブでよい                |
| 黒胡椒       |      |                        |
| オイスターソース  | 大さじ1 | ※ たれ                   |
| にんにく      | 適量   | チューブ ※ たれ              |
| ナンプラー     | 大さじ1 | ※ たれ                   |
| レモン汁      | 大さじ1 | ※ たれ                   |
| スイートチリソース | 大さじ2 | ※ たれ                   |
| 長ネギ       | 適量   |                        |
| パクチー      | 適量   | ぼくは好きなのでどっさり添えるがなくてもよい |

**作り方**
1. 鶏もも肉をトレイにいれ，醤油大さじ1，料理酒大さじ1，生姜適量揉み込み，10 分ほど寝かす
2. 米適量(うちは 3 合以上にすることが多いが，人数が多いから・・)を炊飯器に入れ，分量分水を入れる
3. 炊飯器に上から鶏肉を入れ，黒胡椒をまぶす
4. 炊く
5. 炊けたら鶏肉を取り出し，一口大にカットする．ライスに盛り付けて完成
6. (オイスター醤油だれ) 醤油大さじ1 + オイスターソース大さじ1/2 + にんにくチューブ適量 混ぜる
7. (ナンプラーだれ) ナンプラー大さじ1 + レモン汁大さじ1 混ぜる
8. (スイートチリソース) スイートチリソースそのまま
9. (添え物) パクチーは適当に切って添える．長ネギは白髪ネギにするとよい

**コメント**
* 米は長粒種のほうがあうが，日本米でもふつうに美味しい
* 炊くときの水量は分量分でよい．鶏肉のぶん嵩が増すしとか余計なことは考えないほうがうまくいく
* うちでふだん作るときは家族分もあるので，鶏もも肉 3 枚を炊飯器に敷き詰めているが，とくに不具合はない
* 黒胡椒はこんなもんかなと思う 3 倍くらいかけると美味しい

#### ごぼうの素揚げ

今回唯一のオリジナル？ というかただの手抜き

**材料**

| 品目  | 量  | コメント |
|-----|----|------|
| ごぼう | 1本 |      |
| 岩塩  | 適量 |      |

**作り方**
1. ごぼうを洗って皮をピーラーで取る
2. そのままピーラーで 5~10cm くらいの長さに削る．鰹節削るような要領です．削ったら水につけてあくをとっておく
3. 小鍋にサラダ油を高さ 1cm~ 入れて揚げる準備をする
4. 揚げる．濃いキツネ色になるくらいになったら取り出して盛り付ける．
5. 岩塩振って完成

**コメント**
* 冷蔵庫にごぼうがあるのを見つけると勝手にこれを作るので，豚汁とか作るつもりで買った妻の機嫌には注意する必要がある
