### VCCode snipet

ブログの雛形をいちいち書くのも面倒くさいので，VSCode のスニペット作った．  
参考：
https://qiita.com/xx2xyyy/items/fd333368db548167f15a

こんな感じのものを， xxx.code-snipets(xxx は適当な名前)で作成してやればよい．  
便利．
```json
{
	"blog template": {
		"prefix": ["blog_template", "blog"],
		"body": [
			"---",
			"title: \"\"",
			"description: \"\"",
			"date: $CURRENT_YEAR-$CURRENT_MONTH-$CURRENT_DATE",
			"tags:",
			"\t- Tutorial",
			"\t- GatsbyJS",
			"---",
			"",
			"### subtitle",
			"",
		],
	}
}
```
