#### CD 周り

当 blog は GitHub Actions で雑にビルドしてるのでそっちのサポートも確認する．v19.4.0 ちゃんとあるね．  
https://github.com/actions/setup-node

とりあえず超雑に node を上げる．

```
      - name: setup node
        uses: actions/setup-node@v1
        with:
          node-version: '19.x'
```
