### バイナリは別管理しようねという当たり前気味な話

これまで本ブログで画像ファイルもそのまま GitHub にあげてしまっていたのだが，どうかなぁと思っていたので，別途 images 管理用の S3 と cloudfront を設定してみた．  
記事書いてよさげだったら，ローカルでアップロードとかカチャカチャやるスクリプトも一緒に作った．

たまにしかこういうことをしないので，いつも Mac の sed が BSD 版だということを忘れて無限にハマってしまうので健康によくない．  
いっそ GNU 版いれたほうがいいのかなぁ・・と思いつつ，ひとまず現環境で動くようにしてしまった．
